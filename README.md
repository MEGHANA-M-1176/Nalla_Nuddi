
# Nalla-Nudi 📘

Nalla-Nudi is an offline-first Android application designed as a technical bridge-dictionary for Kannada-medium students. It helps students transition into English-medium higher education by providing simple, contextual Kannada explanations for complex English technical terms in Science, Math, and Commerce.

## 🎯 Problem Statement

Rural and Kannada-medium students often face a significant vocabulary gap when entering English-medium technical education. While they understand the core concepts in their mother tongue, they struggle with English terminology (e.g., *Photosynthesis*, *Trigonometry*, *Osmosis*). Standard dictionaries are often too broad and do not cater to subject-specific needs, leading to reduced confidence in classrooms and interviews.

## ✨ Key Features

*   **100% Offline Capability:** The app works entirely without an internet connection, making it accessible in rural areas.
*   **Contextual Search:** Instantaneously search for English technical terms and get Kannada explanations with simple examples.
*   **Subject Filters:** Browse vocabulary specific to **Science**, **Math**, and **Commerce** using intuitive tabs.
*   **Voice Guide (TTS):** Tap the speaker icon to hear the correct English pronunciation of any term, powered by Android's Text-To-Speech engine.
*   **Interactive Flashcards (My List):** Save difficult words and review them later using a swipeable flashcard interface.
*   **Word of the Day:** Learn a new technical term every day right on the home screen.

## 🛠️ Technology Stack

*   **Language:** Kotlin
*   **Platform:** Android SDK (Minimum SDK 24, Target SDK 34)
*   **Architecture:** Single-Activity Architecture (MVC)
*   **Local Database:** Room Database (SQLite)
*   **Asynchronous Programming:** Kotlin Coroutines
*   **UI/Design:** XML Layouts, Material Design Components, View Binding, ConstraintLayout, ViewPager2 (for flashcards)

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/MEGHANA-M-1176/Nalla_Nuddi.git
   ```
2. Open the project in **Android Studio**.
3. Let Gradle sync and download the necessary dependencies.
4. Connect an Android device via USB (with Developer Options enabled) or start an Android Emulator.
5. Click the green **Run** button to build and install the app.
